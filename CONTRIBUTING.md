# Contributing to FiveOS

Thanks for wanting to help.

## Bugs & feature ideas

Open an [issue](https://github.com/w3bportal/FiveOS/issues). For bugs, include what
happened, what you expected, and the steps to reproduce. Screenshots help a lot.

## Pull requests

Docs fixes under [`docs/`](docs/) and application source changes are welcome via
pull request. Keep changes focused and describe what you changed and why.
