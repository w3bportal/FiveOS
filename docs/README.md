# Welcome to FiveOS

FiveOS is a free Windows app that helps you make FiveM (GTA V multiplayer) mods. No experience needed — each tool walks you through it, one step at a time.


New here? Just find what you want to do below and click it.

## Pick what you want to do:

- Turn a singleplayer car into a FiveM car → [Vehicles](VEHICLES.md)
- Turn a 3D model into a FiveM prop or weapon → [3D Model](3D_MODEL.md)
- Make a big/laggy mod file smaller → [Optimize](OPTIMIZE.md)
- Package a mod into a ready-to-use file → [Mods → RPF](RPF.md)
- Make a custom emote (dance, pose, action) → [Pose → Emote](POSE_TO_EMOTE.md)
- Change how FiveOS works (language, folders, etc.) → [Settings](SETTINGS.md)
